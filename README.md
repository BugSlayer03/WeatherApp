# 🌤️ WeatherNow - Real-Time Weather App

WeatherNow is a responsive and dynamic weather application that provides real-time weather information for any city in the world. It features a clean UI, live weather updates, and dynamically changing backgrounds based on the searched city.

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature display
* 📍 Location and local date
* 🌥️ Weather condition with icons
* 💨 Wind speed information
* 🕒 Last updated time
* 🎨 Responsive and modern UI design

## 🛠️ Tech Stack

* **HTML5**
* **CSS3 (Flexbox & Responsive Design)**
* **JavaScript (ES6)**
* **Weather API** for real-time data

## ⚙️ How It Works

1. User enters a city name in the search bar.
2. The app fetches real-time weather data using a weather API.
3. UI updates dynamically with:

   * Weather details
   * Date and time

## 🔑 Setup Instructions

1. Clone the repository:

   ```
   git clone https://github.com/BugSlayer03/WeatherApp
   ```

2. Add your API keys:

   * Weather API key in `weather.js`

3. Open `index.html` in your browser.
